== Installation

Clone the project :

[source,shell]
[subs=attributes+]
----
cd
git clone {GIT_PROJECT}
----

Add to your `{BASHRC_PATH}` the following code :

[source,shell]
[subs=attributes+]
----
# If the file `{GBW_FILE}` exists, then load it
[[ -f {GBW_PATH} ]] && source {GBW_PATH}
----

Reload `{BASHRC_PATH}` (avoids logout and back to the shell) :

[source,shell]
[subs=attributes+]
----
. {BASHRC_PATH}
----

TADA !
