= Git & Bash Workflow
:numbered:

:BASH_PROFILE_PATH: ~/.bash_profile

== Extend `{BASH_PROFILE_PATH}`

Insert in the end of `{BASH_PROFILE_PATH}` the following code :

[source,shell]
----
if [ -f ~/git-bash-workflow/bash/bash_profile.sh ]; then
    . ~/git-bash-workflow/bash/bash_profile.sh
fi
----
